# Personal Portfolio

This is my personal portfolio website built with **React** and styled using **Bootstrap 5**. It showcases my projects, skills, certificates, and provides contact information.

## Features

- Responsive design with Bootstrap grid and components
- Navigation with React Router for seamless page transitions
- Projects section displaying my main and other projects
- Certificates page highlighting completed and in-progress courses
- Contact form for direct communication
- Light and dark mode toggle (optional)
- Dynamic footer for the year data

## Technologies Used

- React (Functional Components, Hooks)
- React Router DOM
- Bootstrap 5 (CSS framework for responsive layout and styling)
- JavaScript (ES6+)
- HTML5 & CSS3

## Getting Started

1. **Clone this repository**

```bash
git clone https://github.com/Davidva0812/Personal-portfolio.git
cd Personal-portfolio
```

2. **Install dependencies**

```bash
npm install
```

3. **Run the App**

```bash
npm start
```
